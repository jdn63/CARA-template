DHS Scraper Ethics Improvements (Step-by-step)
Overall goal

Make the scraper:

Opt-in (off by default in production)

Polite (rate limiting + retry/backoff)

Efficient (cache with TTL so it doesn’t re-hit DHS constantly)

Transparent (documented behavior + safe defaults)

You’ll implement these mainly in utils/wisconsin_dhs_scraper.py.

Step 0: Decide the default behavior for production

Recommended:

In production (Render): scraper disabled

Use cached data if present; otherwise fallback estimates

Enable scraping only when explicitly set

That means you won’t surprise-load DHS or accidentally scrape on every web request.

def _load_cached_surveillance_data(self, max_age_hours: int = 24):
    """Load cached surveillance data if it exists and is fresh enough."""
    try:
        cache_file = os.path.join(self.cache_dir, "latest_surveillance.json")
        if not os.path.exists(cache_file):
            return None

        mtime = datetime.fromtimestamp(os.path.getmtime(cache_file))
        if datetime.now() - mtime > timedelta(hours=max_age_hours):
            return None

        with open(cache_file, "r") as f:
            return json.load(f)
    except Exception as e:
        logger.warning(f"Could not load cached surveillance data: {e}")
        return None

Step 2 — Add polite requests: User-Agent + delay + retry/backoff

Add import time to your imports at the top:

import time


Then add this method inside the class:

def _request(self, url: str, timeout: int = 15) -> requests.Response:
    """
    Polite request helper with:
    - User-Agent identification
    - baseline delay between requests
    - retry/backoff for transient errors (esp. 429/503)
    """
    delay = float(os.getenv("REQUEST_DELAY_SECONDS", "1.5"))
    max_retries = int(os.getenv("REQUEST_MAX_RETRIES", "3"))

    headers = {
        "User-Agent": os.getenv(
            "SCRAPER_USER_AGENT",
            "CARA-WI-DHS-Scraper/1.0 (contact: github.com/jdn63)"
        )
    }

    for attempt in range(1, max_retries + 1):
        try:
            time.sleep(delay)
            resp = requests.get(url, headers=headers, timeout=timeout)

            if resp.status_code in (429, 503):
                backoff = min(60, 10 * attempt)
                logger.warning(f"{resp.status_code} from {url}. Backing off {backoff}s (attempt {attempt}/{max_retries})")
                time.sleep(backoff)
                continue

            resp.raise_for_status()
            return resp

        except Exception as e:
            if attempt == max_retries:
                raise
            backoff = min(60, 5 * attempt)
            logger.warning(f"Request error for {url}: {e}. Retrying in {backoff}s (attempt {attempt}/{max_retries})")
            time.sleep(backoff)

    raise RuntimeError(f"Failed to fetch {url} after {max_retries} retries")
